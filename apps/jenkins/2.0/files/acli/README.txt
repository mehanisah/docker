Version: 11.3.0-linux-amd64

This is the Appfire Command Line Interface (ACLI) which provides application specific CLIs for popular
Atlassian applications and applications from other providers like Slack, Keen, and Segment. An application CLI
is used to remotely access the application for reporting and automation of application capabilities.
